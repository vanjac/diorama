Extract the following libraries to their respective subdirectories:

- `SDL2/`: [SDL 2](https://www.libsdl.org/)
- `gl3w/`: [gl3w](https://github.com/skaslev/gl3w) (run `gl3w_gen.py` with default settings)
- `glm/`: [glm](https://github.com/g-truc/glm)
- `sketchup/`: [Sketchup SDK](https://extensions.sketchup.com/sketchup-sdk)
